Exame de Faixa Azul

Este catálogo foi desenvolvido como projeto final da matéria Linguagem de Script para Web.

O objetivo é listar técnicas de Jiu-Jitsu com vídeos explicativos ou explicações detalhadas para um melhor aprendizado para quem precisa.

Os videos mostrados no site são apenas ilustrativos, cada academia tem o seu estilo de realizar certos golpes.

Se este site te despertou algum interesse na arte marcial jiu-jitsu considere entrar em contato.
